7.3 Password Strength Check


Algorithm:

1. Input password.

2. Check if length >= 8.

3. Check if contains at least one uppercase letter.

4. Check if contains at least one lowercase letter.

5. Check if contains at least one digit.

6. Check if contains at least one special character (!@#$%^&*()).

7. If all conditions are true, output "strong", else output "weak".

Code:
def check_password(password):
    if (len(password) >= 8 and
        any(ch.isupper() for ch in password) and
        any(ch.islower() for ch in password) and
        any(ch.isdigit() for ch in password) and
        any(ch in "!@#$%^&*()" for ch in password)):
        return "strong"
    else:
        return "weak"

print(check_password("Abc@1234"))  # Output: strong
print(check_password("abid1234"))   # Output: weak
