7.1 Library Fine Calculation


Algorithm:

1. Input the number of late days (days).

2. If days <= 5, fine = days * 2.

3. Else if days <= 10, fine = (5 * 2) + (days - 5) * 3.

4. Else, fine = (5 * 2) + (5 * 3) + (days - 10) * 5.

5. Output the fine.

Program:
A library changes fines based on the number of late days:
1-5 days: 2/day
6-10 days: 3/day
More than 10 days: 5/day

Code:
def calculate_fine(days):
    if days <= 5:
        return days * 2
    elif days <= 10:
        return (5 * 2) + (days - 5) * 3
    else:
        return (5 * 2) + (5 * 3) + (days - 10) * 5

print(calculate_fine(12))  # Output: 35
